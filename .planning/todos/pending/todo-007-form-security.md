# Добавить защиту форм от спама и ботов

Формы сейчас только имитируют отправку (setStatus='success').
Нужно реализовать перед подключением реального бэкенда.

## Требования
- Honeypot-поле (скрытое поле, которое бот заполнит, а человек - нет)
- reCAPTCHA v3 или hCaptcha (невидимая проверка)
- Rate limiting: не более 3 заявок с одного IP в час
- Валидация телефона по маске +7 (XXX) XXX-XX-XX
- CSRF-токен для защиты от подделки запросов
- Защита от XSS в полях ввода (sanitize + escape)

## Примечание
Формы сейчас только в попап-модалах (убраны inline-секции).

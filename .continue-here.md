# Handoff: продолжение работы над sunlife

**Дата паузы:** 2026-07-17
**Последний коммит:** не зафиксирован — все изменения задеплоены
**Статус:** активная разработка, все изменения на сервере

## Что уже сделано (полный список)

### Мобильная версия — исправления
- **Анимация «Почему мамы выбирают нас?»** — мобильная stacking cards анимация через GSAP ScrollTrigger + sticky positioning. Замедлена (180vh на карточку, scrub 1.2), первая карточка поднята ближе к меню (top-16).
- **Слайдер отзывов** — 1 отзыв на мобильном, широкие карточки, свайп пальцем (onTouchStart/onTouchEnd).
- **Кружочки с цифрами** — `inline-flex` вместо `flex` для идеальных кругов.
- **Прейскурант** — название и цена в одну строку на мобильном (`flex-row items-baseline`).
- **Marquee сотрудников** — скорость увеличена с 0.4 до 0.8 px/кадр.

### CTA-система (полная реализация)
- **Mobile Bottom Bar** — sticky glass-панель с 2 кнопками:
  - Primary: контекстная («Забронировать дату», «Стать партнёром» и т.д.)
  - Secondary: иконка `+`, раскрывает меню контактов (Позвонить, WhatsApp, Telegram, VK)
- **Desktop Header CTA** — золотая кнопка справа в шапке
- **Inline CTA** — на всех страницах между секциями:
  - Главная: после Gallery и после PricingCards
  - Цены: после тарифов
  - Портфолио: после галереи
  - Партнёрство: после PartnershipGallery
  - Контакты: форма ContactForm вместо кнопки звонка
  - Политика: ссылка на главную
- **FloatingCallBubble убран** — функционал перенесён в bottom bar

### Формы — компактная версия
- **ContactForm** — уменьшены лейблы (text-sm без капса), отступы, инпуты, кнопки. Модалка max-h-[85vh].
- **PartnershipPopupForm** — аналогично компактнее.
- **InlineCta** — кнопка «Написать» с выпадающим меню мессенджеров.

### Конфиг и рефакторинг
- **site.json** — централизованный конфиг с телефоном, email, WhatsApp, Telegram, VK, текстами CTA.
- **Footer** — использует site.json для ссылок.
- **ContactsPage** — использует site.json, кнопка «Заказать звонок» открывает ContactForm.

## Стек
- Vite 8 + React 19 + TypeScript strict + Tailwind CSS v4
- Framer Motion, GSAP + ScrollTrigger

## Деплой
- Build: `npm run sync-content && npm run build`
- Deploy: `deploy-code.zip` через FTP → `deploy.php?key=sunlife2025deploy`
- Сервер: `sunlife-photo.ru`
